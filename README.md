This project is a decentralized-style web chat application that allows **two users to communicate in real time** through a clean and modern interface.  
The backend is powered by **ASP.NET Core API**, while the frontend is built with **Angular** for a dynamic single-page experience.

---

##  Overview

The goal of **DApp** is to demonstrate a secure and lightweight communication platform where two users can exchange messages instantly.  

### Tech Stack

| Layer | Technology |
|-------|-------------|
| Backend | ASP.NET Core Web API |
| Frontend | Angular |
| Database | SQL Server |
| Communication | SignalR for real-time chat |
| Build Tooling | .NET SDK + Node.js/NPM |

### Prerequisites

Make sure you have installed:

- [.NET 8 SDK or later](https://dotnet.microsoft.com/download)
- [Node.js (v18+)](https://nodejs.org/)
- [Angular CLI](https://angular.io/cli)
